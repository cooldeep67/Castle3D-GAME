# 🏰 Castle Defense 3D

A 3D castle defense game built with **Python**, **PyOpenGL**, and **GLUT**. Take control of an archer defending a medieval castle against endless waves of enemies. Use your bow, protect the castle, and survive as long as possible.

---

## 📖 Overview

Castle Defense 3D is a computer graphics project developed using OpenGL in Python. The game demonstrates 3D modeling, animation, collision detection, camera control, and interactive gameplay mechanics.

---

## ✨ Features

- 🏹 Bow and arrow shooting system
- 👹 Enemy spawning and movement
- 🏰 Fully modeled medieval castle
- 🎥 First-person and third-person camera modes
- ☁️ Animated clouds and dynamic sky
- 🔥 Fire effects
- 🤖 AI-controlled ally archers (God Mode)
- 💥 Arrow physics and collision detection
- 📊 Live game statistics
- 🔄 Restart functionality

---

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/cooldeep67/castle-defense-3d.git
```

### 2. Navigate to the project

```bash
cd castle-defense-3d
```

### 3. Install dependencies

```bash
pip install PyOpenGL PyOpenGL_accelerate
```

### 4. Run the game

```bash
python main.py
```

## 🎮 Controls

| Key | Action |
|------|--------|
| **A** | Rotate Left |
| **D** | Rotate Right |
| **W** | Aim Up |
| **S** | Aim Down |
| **Left Mouse Button** | Shoot Arrow |
| **Right Mouse Button** | Toggle First-Person View |
| **Arrow Keys** | Move Camera |
| **G** | Toggle God Mode |
| **R** | Restart Game |

---

## 🎯 Gameplay

- Defend your castle from incoming enemies.
- Shoot enemies before they reach the castle.
- Every enemy that reaches the castle decreases its health.
- Missing **40 arrows** ends the game.
- The game also ends when **Castle Health** reaches **0**.
- Activate **God Mode** to summon two AI archers that automatically attack enemies.
